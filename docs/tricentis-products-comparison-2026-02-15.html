🔍 SeaLights - 코드 품질의 눈

역할: "이 테스트가 코드의 어디를 커버했나?"

코드 레벨 품질 인텔리전스 플랫폼
주요 기능:
  • 🔍 실행된 테스트가 실제로 어떤 코드를 탔는지 추적
  • 📊 User Story별 코드 커버리지 시각화
  • ⚠️ 미테스트 메서드 자동 감지
  • 🎯 코드 변경 → 영향받는 테스트 자동 선택
  • ⚡ 테스트 사이클 시간 최대 90% 단축
지원 언어:
Java C/C++ Node.js .NET Python JavaScript Go
비유: "코드의 CCTV" - 테스트가 어떤 코드를 건드렸는지 24시간 감시하고 기록

제품 역할 핵심 질문

LiveCompare 분석 (SAP) "SAP 변경이 뭐에 영향 주나?"

겹치는 부분

SeaLights vs LiveCompare: 둘 다 "변경 영향 분석"을 하지만,
• SeaLights: 일반 애플리케이션 (메서드 단위)
• LiveCompare: SAP 전용 (Transport/T-code 단위)

Tricentis의 의도

"한 회사에서 모든 테스팅 니즈를 해결" = 포트폴리오 판매 전략
→ 고객 락인(lock-in) 효과 극대화
