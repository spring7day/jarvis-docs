II. 경쟁사(Tricentis) Agentic AI 전략 현황
2. AI 전략 및 제품 로드맵
Tricentis는 SAP ERP 테스팅 시장에서 3단계 AI 전략을 통해 Agentic AI 선도기업 포지션을 확보 중이며, 품질 에이전트 ARIA를 통해 요구사항 분석 → 테스트 설계 → 실행 → 결과 분석의 End-to-End 자동화를 구현 중 ☞ ARIA의 레코딩 없는 AI 시나리오 자동생성은 PerfecTwin의 핵심 기능 영역과 직접 경쟁 구도 진입
Tricentis AI 3단계 전략
Phase 1
BYOA
(Bring Your Own AI)
고객 보유 AI 모델과 연동 가능한 개방형 아키텍처. AI 전환 진입 장벽 최소화
▶
Phase 2 ★출시완료
AI Agent (ARIA)
(품질 에이전트)
요구사항 분석 → 테스트 설계 → 스크립트 생성 → 실행 → 결과 분석 E2E 자율 수행
▶
Phase 3
AI Workspace
(멀티에이전트, 2026년)
MCP(Model Context Protocol) 기반 복수 에이전트 협력 환경. 2026년 출시 예정
ARIA 품질 에이전트 주요 기능
□
Requirements Intelligence — 자연어 요구사항 → 테스트 시나리오 자동 변환
-
비개발자도 자연어로 테스트 케이스 생성 가능. 테스트 설계 시간 대폭 단축
□
Automated Script Generation — SAP 모듈별 테스트 스크립트 AI 자동 생성
-
SAP FI/MM/SD/PP 표준 T-Code 기반 스크립트 자동 생성. 레코딩 없이 시나리오 구성 가능
□
Risk-based Smart Execution — 변경 영향 분석 기반 테스트 범위 최적화
-
불필요한 테스트 최소화, 핵심 리스크 집중 실행 → 테스트 사이클 타임 단축
SAP 파트너십 현황 및 경쟁 영향
□
SAP Endorsed App 지위 확보 — SAP 공식 마켓플레이스 최상위 파트너 등급
-
SAP Integration Suite, Cloud ALM 네이티브 연동 지원. SAP 생태계 내 완결성 강화
□
S/4HANA Migration 전용 패키지 제공
-
전환 프로젝트 통합 패키지 영업으로 당사 진행 현장(LG전자 N-ERP 등)에서 직접 경쟁 가능성
→
국내 SAP 전환 프로젝트 수주 경쟁에서 글로벌 AI 기능 차이가 선택 요인으로 작용 가능
□
ARIA 데모 성과: SAP FI 구매주문 기반 6단계 E2E 테스트 자동화 구현 시연
-
요구사항 입력부터 결과 분석까지 전 구간 자동화. 실 업무 적용 수준의 완성도 확인
